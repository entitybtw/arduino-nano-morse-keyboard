# Arduino Nano Morse-keyboard

**Arduino Nano Morse-keyboard** — аппаратная клавиатура азбуки Морзе на базе Arduino Nano.

Эта клавиатура является основой моего проекта, представляемого на  
VIII региональном конкурсе исследовательских и проектных работ школьников  
**«Высший пилотаж — Пенза 2026»**.

Веб-версия тренажёра для работы с клавиатурой: **https://morse.entitybtw.ru**

---

## Особенности

- Бюджетная сборка — собрать клавиатуру можно недорого  
- Раскладки клавиатуры (смена раскладки при долгом зажатии кнопки):  
  - Английская  
  - Русская  
  - Морзе  

---

## Зависимости

1. **unoHID** — эмуляция клавиатуры  
2. **NewTone** — улучшенная библиотека для воспроизведения звука  

Все библиотеки доступны в папке `libs` в корневой директории репозитория.
